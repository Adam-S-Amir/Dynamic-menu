# Dynamic-menu
Upscale your website menu's with this purely JS API!

```
            <div id="inner-menu" class="inner-menu">
                <script src="./Start-Menu-1.js"></script>
                <script src="./Start-Menu-2.js"></script>
                <script src="./Start-Menu-3.js"></script>
                <table cellpadding="0" cellspacing="0">
                    <tbody>
                        <tr>
                            <td style="border-left:solid 1px #DFDFDF;">
                                <script src="./Start-Menu-Styles-1.js"></script>
                            </td>
                        </tr>
                    </tbody>
                </table>
                <script src="./Start-Menu-Styles-2.js"></script>
            </div>
```

npm

```
npm install @magnusmarks/dynamic-menu
```
